Listar Pedidos - UC012

O usuário terá em sua tela uma lista de pedidos efetuados por compradores.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema;
  • O vendedor deve estár disponível;
  • O comprador deve ter realizado uma compra;
  • O vendedor deverá ter produtos cadastrados.

Pós-condições:

  • Uma lista de pedidos deverá aparecer.
