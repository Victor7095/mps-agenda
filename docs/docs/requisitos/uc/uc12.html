Visualizar Calndário - ECU012

Mostra ao usuário um calendário interativo que exibe eventos e tarefas que irão acontecer no dia especificado.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema.

Pós-condições:

  • A tela exibe os itens do dia especificado.
