Convidar pessoas - ECU004

O vendedor após ter sido cadastrado no sistema, tem a opção de cadastrar os seus produtos para venda, onde será cadastrado o nome do produto, valor.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema.

Pós-condições:

  • O usuário terá adicionado outros usuários à reunião, e deverá prosseguir com o cadastro da reunião.
