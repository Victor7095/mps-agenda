Notificar Usuário - UC010

Depois da compra do produto, tanto o comprador quanto o vendedor receberá uma notificação da efetivação da compra.

Ator:

  • Usuário

Pré-condições:

  • Precisa estar logado;
  • O usuário precisa ter escolhido a forma de retirada;
  • Precisa ter comprado um produto.

Pós-condições:

  • Usuário terá o produto.
