Visualizar Notificações - ECU010

Mostra ao usuário convites de reuniões e tarefas próximas de acontecer.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema, um sininho no canto superior direito deve aparecer, informando ao usuário uma notificação nova.

Pós-condições:

  • Depois que o usuário visualizar todas as notificações o sininho desaparece.
