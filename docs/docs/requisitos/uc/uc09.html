Finalizar compra - UC009

O usuário deverá finalizar a compra do produto.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema;
  • O usuário deve ter escolhido algum produto;
  • O usuário deve ter preenchido a quantidade e escolhido o local de entrega.

Pós-condições:

  • O usuário terá o produto em mãos
