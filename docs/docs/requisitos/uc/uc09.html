Excluir Reunião - ECU009

Exclui uma reunião que foi adicionada anteriormente.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema, a reunião tem que estar salva no sistema e deve visualizada pelo ator.

Pós-condições:

  • A reunião é excluída da conta do usuário.
