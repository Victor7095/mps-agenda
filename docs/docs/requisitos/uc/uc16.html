Listar Produtos do Vendedor - UC016

O vendedor pode listar os produtos que foram cadastrados no sistema.

Ator:

  • Usuário

Pré-condições:

  • O usuário precisa estar logado no sistema como vendedor.
  • É necessário que tenha cadastrado algum produto.

Pós-condições:

  • O vendedor pode editar seus produtos cadastrados.
