Excluir Tarefa - ECU008

Exclui uma tarefa que foi adicionada anteriormente.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema, a tarefa tem que estar salva no sistema e deve visualizada pelo ator.

Pós-condições:

  • A tarefa é excluída do sistema.
