FEscolher Forma de Retirada - UC008

O comprador poderá escolher como o irá adiquirir o produto: o vendedor se deslocará até o comprado ou vice-versa..

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema;
  • O usuário deve ter escolhido algum produto

Pós-condições:

  • O usuário adiquirirá o produto.
