Recuperar Senha - UC003

Caso o usuário tenha esquecido a senha, ele será capaz de recuperar o acesso ao sistema criando outra senha.

Ator:

  • Usuário

Pré-condições:

  • Usuário precisa estar na página de login;
  • Precisa ter o email cadastrado no momento da criação da conta.

Pós-condições:

  • O usuário terá acesso ao sistema novamente.
