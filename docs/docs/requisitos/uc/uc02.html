Cadastrar Usuário - UC002

O ator deve poder cadastrar um usuário no sistema.

Ator:

  • Usuário (vendedor, comprador)

Pré-condições:

  • O ator estar na página de login do sistema;
  • O ator acessar o cadastrado no sistema;
  • O ator digitará os dados necessários para o cadastro.

Pós-condições:

  • O usuário deverá estar cadastrado no sistema.
