Editar Perfil - UC011

O usuário pode alterar suas informações caso queira.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema;
  • O usuário deverá ter as informações no sistemas.

Pós-condições:

  • Alguma informação do usuário poderá ter sido alterada.
