Avaliar Pedido - UC017

O usuário deverá avaliar a qualidade do pedido, levando em conta o produto e a qualidade da venda (Tempo de entrega, bom atendimento), avaliando de 1 a 5 estrelas.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema
  • O usuário deve ter efetuado e finalizado uma compra

Pós-condições:

  • O usuário poderá efetuar uma nova compra
  • O usuário poderá utilizar qualquer funcionalidade do sistema
