Visualizar Listas - ECU014

Objetiva mostrar ao usuário todas as listas de compras que ele cadastrou no app.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema.

Pós-condições:

  • A lista de listas é carregada normalmente.
