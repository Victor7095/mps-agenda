Descrição do Pedido - UC014

O vendedor poderá ver informações referente ao pedido efetuado pelo comprador.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema
  • O usuário deve ter finalizado um pedido;
  • O vendedor deve estar disponível;
  • O vendedor deve ter aceitado a compra.

Pós-condições:

  • O comprador irá receber o produto do vendedor no local escolhido.
