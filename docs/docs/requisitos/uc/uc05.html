Exibir Produtos - UC005

O sistema mostrará uma lista de produtos que o usuário poderá comprar.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema;
  • Deve haver algum produto cadastrado no banco de dados;
  • Algum usuário deve estar disponível para vender seus produtos.

Pós-condições:

  • O usuário poderá visualizar as informações do produto;
  • O usuário poderá pedir a entrega do produto.
