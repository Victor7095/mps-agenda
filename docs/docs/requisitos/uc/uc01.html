Logar Usuário - UC001

O ator deverá efetuar login no sistema.

Ator:

  • Usuário (vendedor, comprador)

Pré-condições:

  • O ator deve estar na página de login do sistema;
  • O ator deve estar cadastrado no sistema.

Pós-condições:

  • O usuário estará logado no sistema.
