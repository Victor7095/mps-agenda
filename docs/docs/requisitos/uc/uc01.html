Fazer Login - ECU001

Objetiva validar e autenticar o nome/e-mail e a senha do ator usuário para que ele se conecte à aplicação .

Ator:

  • Usuário

Pré-condições:

  • O ator deve estar cadastrado no sistema.

Pós-condições:

  • O ator fica habilitado a utilizar a parte restrita da aplicação.
