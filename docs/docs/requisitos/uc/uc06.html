Escolher Produtos - UC006

O ator vai poder selecionar o produto que deseja.

Ator:

  • Usuário

Pré-condições:

  • O Usuário deve estar logado no sistema.
  • Deve haver algum produto cadastrado por um vendedor.
  • O Usuário deve estar na lista de produtos.

Pós-condições:

  • Deverá aparecer a descrição do produto.
