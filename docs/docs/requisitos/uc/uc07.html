Excluir Lista- ECU007

Objetiva validar o processo de exclusão de lista.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema;
  • Uma lista deve estar registrada no sistema.

Pós-condições:

  • O usuário terá excluído uma lista.
