Escolher Quantidade - UC007

O usuário pode escolher quantas unidades quiser daquele determinado produto.

Ator:

  • Usuário

Pré-condições:

  • O usuário deve estar logado no sistema;
  • O usuário deve ter selecionado um produto.

Pós-condições:

  • O usuário pode finalizar a compra.
