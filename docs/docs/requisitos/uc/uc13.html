Estar Disponível - UC013

O usuário precisa estar disponíveis para efetuar procedimentos de venda.

Ator:

  • Usuário

Pré-condições:

  • O usuário precisa estar logado.

Pós-condições:

  • O usuário estará disponível para fetuar procedimentos.
