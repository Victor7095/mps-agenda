Visualizar Reuniões - ECU013

Objetiva mostrar ao usuário todas as reuniões nas quais ele irá participar e os detalhes de uma determinada reunião selecionada.

Ator:

  • Usuário

Pré-condições:

  • O usuário precisa estar logado.

Pós-condições:

  • A lista de reuniões é carregada normalmente.
