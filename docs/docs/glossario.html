Glossário Cronoz

Apresentar, como referencial, definições de termos usualmente empregados pelo sistema desenvolvido. Afim de facilitar o entendimento do mesmo.

EST: Unidade de Tecnologia da UEA, EST significa Escola Superior de Tecnologia. Localizada na Av.Darcy Vargas, 1200.

Home ou tela Home: É a tela principal, por onde o usuário tem acesso a todas as funcionalidades principais do sistema. Tela apresentada após o login do usuário.

Kick-off: em inglês, que signfica ponto de partida ou começo. O Kick-off são os requisitos inicias que devem ser atendidos para dar sequência no processo de Modelagem e Projeto de Sistemas.

Protótipo: é o termo usado para se referir ao que foi criado pela primeira vez, servindo de modelo ou molde para futuras produções.

Sistema: É a aplicação da qual o usuário vai interagir.

UEA: Universidade Do Estado do Amazonas, faculdade pública do Amazonas.

UML: é um acrônimo para a expressão Unified Modeling Language. Pela definição de seu nome, vemos que a UML é uma linguagem que define uma série de artefatos que nos ajuda na tarefa de modelar e documentar os sistemas orientados a objetos que desenvolvemos.

Usuário: ator ou pessoa que vai interagir com o sistema proposto, no caso Cronoz. O usuário pode ser tanto vendedor quanto comprador.
